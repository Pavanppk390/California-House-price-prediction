# California Housing Price Prediction

This project uses Linear Regression to predict California house prices using the California Housing dataset from Scikit-Learn.

## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Multicollinearity check using VIF
- Feature scaling
- Linear Regression model training
- Model evaluation
- Model serialization using Pickle

## Results

R² Score: 0.5868

MAE: 0.5434

MSE: 0.5378

RMSE: 0.7333

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
- Statsmodels
